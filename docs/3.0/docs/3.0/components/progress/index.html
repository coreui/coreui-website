Bootstrap progress

Bootstrap progress bars component can be used for showing the progress of a task or action to the users. Bootstrap progress supports for stacked bars, animated backgrounds, and text labels. Learn how to create different types of progress bars.

On this page

    • Labels

How it works

Progress components are built with two HTML elements, some CSS to set the width, and a few attributes.

The .c-progress has to be used as a wrapper to indicate the max value of the progress bar. Inside you have puy .c-progress-bar element to indicate the progress so far. The .c-progress-bar needs an inline style, utility class, or custom CSS to set their width and also needs some role and aria attributes to make it accessible.

Please check following examples.

<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" aria-valuenow="0" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" style="width: 25%" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" style="width: 50%" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" style="width: 75%" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" style="width: 100%" aria-valuenow="100" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Bootstrap’s CSS provides a handful of utilities for setting width. Depending on your requirements, these may help with quickly configuring progress.

<div class="c-progress">
  <div class="c-progress-bar c-w-75" role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Labels

Attach labels to your progress bars by putting text within the .c-progress-bar.

25%
<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" style="width: 25%;" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100">25%</div>
</div>

Height

Bootstrap only set a height value on the .c-progress, so if you modify that value the inner .c-progress-bar will automatically resize.

<div class="c-progress" style="height: 1px;">
  <div class="c-progress-bar" role="progressbar" style="width: 25%;" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress" style="height: 20px;">
  <div class="c-progress-bar" role="progressbar" style="width: 25%;" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Backgrounds

You can use background utility classes to modify the look of individual progress bars.

<div class="c-progress">
  <div class="c-progress-bar c-bg-success" role="progressbar" style="width: 25%" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-bg-info" role="progressbar" style="width: 50%" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-bg-warning" role="progressbar" style="width: 75%" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-bg-danger" role="progressbar" style="width: 100%" aria-valuenow="100" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Multiple bars

Combine progress bars in a progress component if you want.

<div class="c-progress">
  <div class="c-progress-bar" role="progressbar" style="width: 15%" aria-valuenow="15" aria-valuemin="0" aria-valuemax="100"></div>
  <div class="c-progress-bar c-bg-success" role="progressbar" style="width: 30%" aria-valuenow="30" aria-valuemin="0" aria-valuemax="100"></div>
  <div class="c-progress-bar c-bg-info" role="progressbar" style="width: 20%" aria-valuenow="20" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Striped

Add .c-progress-bar-striped to any .c-progress-bar to apply a stripe via CSS gradient over the progress bar’s background color.

<div class="c-progress">
  <div class="c-progress-bar c-progress-bar-striped" role="progressbar" style="width: 10%" aria-valuenow="10" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-progress-bar-striped c-bg-success" role="progressbar" style="width: 25%" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-progress-bar-striped c-bg-info" role="progressbar" style="width: 50%" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-progress-bar-striped c-bg-warning" role="progressbar" style="width: 75%" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
</div>
<div class="c-progress">
  <div class="c-progress-bar c-progress-bar-striped c-bg-danger" role="progressbar" style="width: 100%" aria-valuenow="100" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Animated stripes

The striped gradient can also be animated. Add .c-progress-bar-animated to .c-progress-bar to animate the stripes right to left via CSS3 animations.

<div class="c-progress">
  <div class="c-progress-bar c-progress-bar-striped c-progress-bar-animated" role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100" style="width: 75%"></div>
</div>
